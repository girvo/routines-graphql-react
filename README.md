# my-routines

## Development

- `nvm use v24.8.0` (or install it manually)
- `corepack enable`
- `pnpm install`
- `pnpm dev` to start backend, frontend, storybook via `concurrently`
- `pnpm dev:split` to start backend, frontend, storybook in a Tmux split window
  -  `Ctrl-b` - `:kill-session` to stop the tmux session

## VSCode

- Install the extensions this asks for
- Cmd-shift-p `Relay: Start compiler` (if it didn't start automatically, which it should've!)
